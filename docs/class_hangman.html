Detailed Description

Модуль игры Висилица.

Этот модуль содержит основную логику игры "Висилица".

Member Function Documentation

◆ check_records()

void Hangman::check_records ( )

Проверка очков игрока на попадение в таблицу рекордов.

При завершении игры(закрытие формы, достигнуто максимальное число ошибок, все слова отгаданы) происходит проверка текущих очков игрока. Если он набрал достаточное количество очков, ему будет предложено вписать свое имя в специальную форму, после черо проризойдет обновление таблицы рекордов.
